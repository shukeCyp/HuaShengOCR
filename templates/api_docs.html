API 文档

这是给前端/集成方看的页面版接口文档。Swagger 文档也可用。

打开 Swagger UI · 查看 OpenAPI JSON

1. 健康检查

GET /health
{
  "ok": true,
  "engine": "OnnxOCR"
}

2. 单图 OCR

POST /api/ocr
上传一张图片，返回 OCR 文本与坐标框。
curl -X POST /api/ocr \
  -F "file=@demo.jpg"

3. 视频单帧检查

POST /api/video-review/frame-check
适用于浏览器本地抽帧模式：每秒抽一帧，再把当前帧发给后端判断是否包含链接。
curl -X POST /api/video-review/frame-check \
  -F "file=@frame_001.jpg" \
  -F "second=12"
{
  "success": true,
  "passed": false,
  "second": 12,
  "text": "访问 www.example.com",
  "links": ["www.example.com"],
  "matched": true,
  "count": 1,
  "items": []
}

4. 后端整段视频审核任务

POST /api/video-review/tasks
上传整段视频，交给后端每秒抽帧处理。当前页面主流程已改为浏览器本地抽帧，但这个接口仍保留。
